DevConnector

A small social network designed for programmers. Register, create a profile, write posts, comment on posts and delete account. User authentication with Passport and JSON Web Tokens.

Carbon Neutral

Provides environmental impact models for daily activities based on user input. Designed for the environmentally conscious. View search history and delete items. API integration.

Onion Scraper

View articles scraped from The Onion Politics and leave personal notes. Uses Axios, Cheerio and Mongoose.

Liri Bot

Allows user to search for concert, song, or movie info. Uses Node. Search for specific concert, song or movie info from the Command Line. API integration with Axios.

GifTastic

Search for gifs from GIPHY and save favorites. API integration and AJAX.

Friend Finder

Find your closest friend match by comparing user input to a friends list stored serverside. Uses RESTful APIs.
